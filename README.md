# 🎂 Happy Birthday Sir
**Dr. Rupam Kumar Sharma**

A premium Apple-inspired, Krishna-conscious birthday tribute website dedicated to **Dr. Rupam Kumar Sharma**, Assistant Professor, Department of Computer Science & Engineering, Rajiv Gandhi University.

This project combines modern UI/UX principles, smooth animations, spiritual aesthetics, and responsive web design to create a memorable digital birthday experience.
